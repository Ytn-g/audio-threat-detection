# Real-Time Audio Threat Detection

> Edge AI system that detects threatening sounds (glass breaking, alarms, gunshots)
> on a Raspberry Pi 4 with <150ms latency, using multimodal audio + vision fusion
> to suppress false positives.

## Stack

`Python` `TensorFlow Lite` `PyAudio` `librosa` `OpenCV` `MQTT` `systemd` `Raspberry Pi 4`
